Campo de Data

São elementos muito comuns em formulários. Sua função é fornecer suporte ao usuário para a inserção de datas e reduzir o grau de propensão a erros.

As características desejáveis são:

  • Distinção visual no input que indique a presença de recurso acessório;
  • Boa navegabilidade entre meses e anos;
  • Distinção visual bem destacada para o dia selecionado;
  • Atalho para o "dia de hoje";
  • Flexibilidade técnica para formatos diversos;

O exemplo abaixo foi construído com a biblioteca JQueryUI com o propósito de ilustrar o componente Campo de Data

Código do exemplo:

    <label for="datepicker">Data:</label>
      <div class="input-group date">
        <input type="text" class="form-control" id="datepicker">
        <span class="input-group-addon">
          <i class="fa fa-calendar"></i>
        </span>
      </div>

      <script type="text/javascript">
