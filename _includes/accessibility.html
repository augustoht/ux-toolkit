Considerações sobre Acessibilidade

O uso de tecnologias assistivas é essencial para tornar o sistemas acessíveis a pessoas portadoras de necessidades especiais. Além disso, é uma recomendação que faz parte do Programa de Governo Eletrônico do Ministério do Planejamento, Orçamento e Gestão.

O programa disponibiliza uma série de documentos contendo diretrizes para a construção de sites e aplicativos web. Esta suíte inclui recomendações sobre usabilidade, acessibilidade, redação, codificação, manutenção etc.

Nos últimos anos, o Consórcio W3C, ciente da evolução dos sistemas web e o surgimento de elementos visuais cada vez mais ricos e complexos, lançou uma iniciativa para especificar recursos de acessibilidade a serem utilizados em aplicações de interface rica construídas em Ajax, HTML, Javascript e outras tecnologias relacionadas.
